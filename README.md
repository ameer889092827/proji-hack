# Платформа для координации волонтеров (AI Matching)

Это full-stack веб-приложение, которое помогает кураторам создавать задачи для волонтеров, а системе — автоматически подбирать наиболее подходящих кандидатов с помощью искусственного интеллекта (Google Gemini API).

## 🛠 Технологии
- **Frontend:** React 18, Vite, Tailwind CSS, Framer Motion, Lucide React
- **Backend:** Node.js, Express
- **AI:** Google Gemini API (`@google/genai`)

## 🚀 Как запустить проект у себя локально

### 1. Предварительные требования
Убедитесь, что у вас установлены:
- [Node.js](https://nodejs.org/) (рекомендуется версия 18 или выше)
- npm (устанавливается вместе с Node.js)

### 2. Установка зависимостей
Откройте терминал в папке с проектом и выполните команду:

```bash
npm install
```

### 3. Запуск сервера для разработки
После установки зависимостей запустите проект командой:

```bash
npm run dev
```

Сервер запустится, и приложение будет доступно в браузере по адресу:
👉 **http://localhost:3000**

### 4. Настройка API-ключа Gemini
В этом проекте **не нужно** создавать файл `.env`. 
При первом открытии приложения в браузере появится стартовый экран, где вам нужно будет ввести ваш **Gemini API Key**. 
- Ключ сохранится в памяти вашего браузера (`localStorage`).
- При каждом запросе к AI он будет безопасно передаваться на сервер через HTTP-заголовки.
- Вы всегда можете изменить ключ, нажав кнопку «Сменить ключ» в правом верхнем углу на экране выбора роли.

Получить бесплатный API-ключ можно в [Google AI Studio](https://aistudio.google.com/app/apikey).

## 📁 Структура проекта
- `server.ts` — Express-сервер, обрабатывающий API-запросы к Gemini (`/api/parse-task`, `/api/match-volunteers`, `/api/chat`).
- `src/App.tsx` — основной React-компонент, содержащий всю логику интерфейса (экраны куратора, волонтера, чат и т.д.).
- `src/index.css` — глобальные стили и конфигурация Tailwind CSS.
